# 開発ルール

本プロジェクトにおけるエージェントの基本ルールは以下の通りです。

- 会話、コード内へのコメント、GitHub のコミットコメントなど、すべて **日本語** を基本としてください。
- ワークスペースのルートにある本ファイル（.agent/rules.md）の内容を常に遵守してください。
- サイトのフロントエンドは「シンプルで落ち着いた、お洒落なデザイン」を心がけ、モバイル表示（レスポンシブ）に完全対応すること。
- レンタルサーバー（さくらインターネット）の制約により、追加のアプリケーションやミドルウェアのインストールができないため、標準で動作する言語（PHP等）を使用して構築すること。
